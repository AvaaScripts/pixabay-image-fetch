# Pixabay Image Fetch

A Node.js web application that fetches images from the Pixabay API and displays them in a gallery.

## Features
- Search and display images from Pixabay
- Responsive image gallery
- Simple and clean UI

## Technologies Used
- Node.js
- Express.js
- EJS (Embedded JavaScript Templates)
- Fetch API
